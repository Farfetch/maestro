Environment Variablesâ€‹

Maestro application services could be configured by using the Environment variable. There are two ways of passing the configuration to the service:

  • Create .env file in the project root directory
  • Make variables available in the environment where the application starts. There are a lot of ways how you define and use them, here is the link to How to Set and List Environment Variables in Linux.
